== Description ==

Provides a setting to show or hide the new admin bar in <strong>Wordpress 3.1</strong>

== Installation ==

You can use the built in installer and upgrader, or you can install the plugin manually.

**Installation via Wordpress**

1. Go to the menu 'Plugins' -> 'Install' and search for 'Show/Hide Adminbar'
1. Click 'install'

**Manual Installation**

1. Upload folder `showhide-adminbar` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure plugin in 'Settings -> Admin Bar' menu
